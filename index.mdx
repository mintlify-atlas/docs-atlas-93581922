Gleam is a statically-typed functional programming language that brings the reliability of a strong type system to the Erlang ecosystem. Write once, run on the battle-tested BEAM virtual machine or compile to JavaScript — all with friendly error messages that actually help you fix bugs.

## Why Gleam?

Gleam combines the power of a strong static type system with the concurrency and fault-tolerance of the Erlang ecosystem. It's designed to be approachable for developers coming from any background.

<CardGroup cols={3}>
  <Card title="Type safe" icon="shield-check">
    Full type inference means fewer annotations but total type safety — no runtime type errors.
  </Card>
  <Card title="Runs everywhere" icon="globe">
    Compile to Erlang/BEAM for scalable servers, or to JavaScript for browsers and Node.js.
  </Card>
  <Card title="Friendly errors" icon="face-smile">
    Error messages are clear, precise, and suggest fixes — so you spend less time debugging.
  </Card>
  <Card title="No nulls" icon="ban">
    Gleam has no null values. Use `Option` and `Result` types to handle absence and errors explicitly.

## Get started

<Steps>
  <Step title="Install Gleam">
    Install the Gleam compiler using your preferred method. See the [installation guide](/installation) for all options.

    ```bash
    brew install gleam
    ```
  </Step>
  <Step title="Create a new project">
    Use `gleam new` to scaffold a new project with sensible defaults.

    ```bash
    gleam new my_app
    cd my_app
    ```
  </Step>
  <Step title="Run your project">
    Build and run your project with a single command.

    ```bash
    gleam run
    ```
  </Step>
  <Step title="Explore the language">
    Dive into the [Language Guide](/language/basics) to learn Gleam's syntax and features, or browse the [CLI Reference](/cli/overview) for all available commands.
  </Step>
</Steps>

## Example

Here's a taste of what Gleam code looks like:

```gleam
import gleam/float
import gleam/io
import gleam/list

pub type Shape {
  Circle(radius: Float)
  Rectangle(width: Float, height: Float)
}

pub fn area(shape: Shape) -> Float {
  case shape {
    Circle(radius) -> 3.14159 *. radius *. radius
    Rectangle(width, height) -> width *. height
  }
}

pub fn main() {
  let shapes = [Circle(5.0), Rectangle(3.0, 4.0)]
  list.each(shapes, fn(shape) {
    io.println("Area: " <> float.to_string(area(shape)))
  })
}
```

<Tip>
  Gleam's `case` expressions are exhaustive — the compiler ensures you handle every possible variant of a type.
</Tip>
